My Diary

Monday January 18th, 2021

I started my internship with Manifest & Company officially today. Being the first day of work, I started with the basics - working mostly with HTML files.
The CSS templates using Bootstrap and Javascript files were given in the resources for learning today. I learnt how to properly organise my files in specific folders.
Basically, I had folders for each of the different file types: HTML, CSS, Javascript and images. Properly organising the files in folders enables easy location and referencing, especially when including them in anchor tag href attribute, image source links, javascript source attribute as well as CSS href attribute.
The webpage I worked with today consists of five pages: Home, About Me, Portfolio, My Diary and Contact. I learnt how to link the pages to each other, and set a current page being viewed as "active".
Finally, I learnt how to freely host my project using Github's free hosting service
The task I have for myself after completing the day's work is to visit bootstrap's site and learn how to use the bootstrap css contents in my webpage when and where I need them.
